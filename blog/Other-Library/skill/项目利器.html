# IOS键盘字母输入,默认首字母大写的解决方案

设置如下属性

<input autocapitalize="off" autocorrect="off" />

input的三个属性autocomplete:默认为on,代表是否让浏览器自动记录输入的值,可以在input中加入autocomplete="off"来关闭记录,保密输入内容;autocapitalize:自动大小写;autocorrect:纠错
