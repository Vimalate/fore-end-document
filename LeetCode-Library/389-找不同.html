# 找不同

给定两个字符串 s 和 t，它们只包含小写字母。

字符串 t 由字符串 s 随机重排，然后在随机位置添加一个字母。

请找出在 t 中被添加的字母。

 

示例:

输入：
s = "abcd"
t = "abcde"

输出：
e

解释：
'e' 是那个被添加的字母。

# 解(1)：

var findTheDifference = function (s, t) {
    s = s.split('').sort()
    t = t.split('').sort()
    for (let i = 0; i < t.length; i++) {
        if (s[i] !== t[i]) {
            return t[i]
        }
    }
};

# 解（2）：

var findTheDifference = function (s, t) {
    // s 中元素是否存在于 t 中，在则替换为''
    for (let item of s) {
        t = t.replace(item, '')
    }
    return t
};
